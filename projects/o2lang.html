🔧 About the Language

O₂lang is a lightweight, real-time compiler-based programming language, designed for simplicity, portability, and customization. It's perfect for small projects, scripting, bots, and personal automation.

⚙️ Core Features

Python Engine

Built on Python for cross-platform compatibility

Custom Syntax

Customize syntax via syntax.lang file

CLI Tools

o2 init, o2 build, o2 build --release

🧪 Sample Code

let name = "Unknownmsv"
fn greet(user):
    print("Hi " + user)

greet(name)

🚀 Getting Started

install O₂lang

$ pip install o2lang

$ o2 --version
